## Beschrijving
 De opdrachtgever is Tumi Mundo. TuMi Mundo heeft een app die ouders helpt met het tweetalig opvoeden van hun kind op basis van wetenschappelijk onderzoek.. Tumi mundi is gericht op ouders met kinderen van 0-4 jaar, waarmee de kinderen makkelijk de taal kunnen op pakken. De web app bestaat uit verhalen die je kan beluisteren in verschillende talen. ## Installatie

2. Clone of download deze respository.
3. Open hem in je code-editor (Visal Studio Code).
4. Open de terminal en typ npm install.
